Admin Initial Setup

To continue your setup please follow the directions:

Step One: Fill out Environment Variables in Heroku Settings Page

Rquired

ADMIN_PASSWORD : your-master-password

AWS_ACCESS_KEY_ID : your-aws-access-key

AWS_SECRET_KEY : your-aws-secret-key

AWS_DOCUMENTS_BUCKET_NAME : your-aws-document-bucket-name

AWS_PROFILE_IMAGES_BUCKET_NAME: your-aws-images-bucket-name

Optional

INFURA_URI : your-infura-uri

ETHER_GAS_STATION_API : https://ethgasstation.info/api/ethgasAPI.json

ETH_FUNDING_PRIVATE_KEY : your-private-key-that-funds-notarizations

ETH_MYPASS_DOMAIN_PRIVATE_KEY : your-private-key-that-owns-your-ens-domain
